# kotlin-cli-gradle-crypto-aes-password

## Description
A demo for AES encryption of a password.
AES is a 256 byte encryption.

## Tech stack
- kotlin
- gradle

## Docker stack
- gradle:jdk11

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
